## Task 1: Exploring and Visualizing the Iris Dataset

### Objective
The goal of this task was to load the classic Iris dataset and use different types of graphs to understand the data patterns.

### Key Insights from My Analysis
* **Scatter Plot:** I noticed that the Setosa species forms a completely separate group from the other two species, which makes it very easy to identify based on sepal sizes.
* **Histogram:** The petal length distribution shows two distinct peaks, meaning the flowers clearly fall into different size categories.
* **Box Plot:** Virginica consistently has the largest petal width, while Setosa has the smallest and most consistent size with very little variation.

## Task 2: Credit Risk Prediction

### What I did:
For this task, I generated a dataset of loan applicants containing features like Income, Loan Amount, Education, and Credit History. Then, I trained a Logistic Regression model to predict if a loan should be approved or not.

### My Observations:
* I plotted a count chart which showed that graduate applicants generally get more loan approvals.
* Credit history is the most important factor because applicants with a good past score get high approval rates.
* The machine learning model trained successfully on the data and gave a very good accuracy score for predicting loan risks.
